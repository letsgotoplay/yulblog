best practices found

Configuaration

All client related objects & beans is register by Auto-configuration class. and All configuration class is picked by annotation/conditional class in class path.

Package info

annotation on package to force all classes in packge to have no null api and fields.

Customizer with builder patthern

Marker Class as enable/disable Flag

Instance

private constructor, Apply method to instantiate Object. every modification make sure new object is created, and let GC to do the work. instead of manipulation of old ones.

All the complexity is hide in private apply method. and expose out simple overload methods for use.
